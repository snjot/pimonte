# pimonte

A program for approximate calculation of pi using the Monte Carlo method.

## Requirements

- Python version 3.6+

## Usage

```zsh
git clone git@github.com:snjot/pimonte.git
cd pimonte
```

You can calculate pi approximation as follows:

```zsh
% python -m pimonte
pi = 3.1413028; (1.8547039031982422 sec)
```
